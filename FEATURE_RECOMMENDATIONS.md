# 功能增强建议

## 🎯 优先级分类

### 🔴 高优先级（强烈推荐）

这些功能能显著提升用户体验和学习效果。

#### 1. **对话录音和回放** ⭐⭐⭐⭐⭐

**价值**：
- 用户可以回顾自己的发音和表达
- 对比不同会话的进度
- 用于自我评估和改进

**实现复杂度**：⭐⭐ (中等)

**技术方案**：
```
前端：
- 使用 MediaRecorder API 录制麦克风音频
- 保存为 WebM 或 MP3 格式
- 上传到后端存储

后端：
- 创建 audio_recordings 表
- 存储音频文件到 S3 或本地存储
- 提供音频下载接口

数据库：
- ScenarioDialogueSession 添加 audioUrl 字段
```

**预期效果**：
- 用户可以听回自己的对话
- 更好地理解发音问题
- 增加用户粘性

---

#### 2. **实时发音评测** ⭐⭐⭐⭐⭐

**价值**：
- 实时反馈用户发音质量
- 帮助用户立即改正
- 提升发音学习效果

**实现复杂度**：⭐⭐⭐ (中高)

**技术方案**：
```
集成 Azure Speech Services 或 Google Cloud Speech-to-Text：

前端：
- 在对话中实时发送音频样本
- 显示发音评分（0-100）
- 显示音素级别的反馈

后端：
- 调用 Azure Speech API 进行发音评估
- 返回详细的发音评分
- 存储发音数据用于分析

系统提示增强：
- 根据发音评分动态调整 AI 反馈
```

**预期效果**：
- 用户能立即知道发音是否正确
- 显著提升发音学习效果
- 差异化的学习体验

#### 4. **自适应难度调整** ⭐⭐⭐⭐

**价值**：
- 根据用户能力自动调整难度
- 避免过简单或过难
- 优化学习效果

**实现复杂度**：⭐⭐⭐ (中高)

**技术方案**：
```
算法：
- 基于最近 5 次会话的平均评分
- 如果平均评分 > 80，建议升级难度
- 如果平均评分 < 60，建议降级难度

实现：
- 在场景选择页面显示推荐难度
- 在会话完成后给出难度建议
- 记录用户的难度进度

系统提示调整：
- 根据难度动态调整 AI 的语速和词汇复杂度
```

**预期效果**：
- 用户始终处于最优学习区间
- 学习效率提升 20-30%
- 用户满意度提高

```typescript
const mediaRecorder = new MediaRecorder(stream);

mediaRecorder.ondataavailable = (e) => chunks.push(e.data);
mediaRecorder.onstop = async () => {
  const blob = new Blob(chunks, { type: 'audio/webm' });
  const formData = new FormData();
  formData.append('audio', blob);
  formData.append('sessionId', sessionId);
  await fetch('/api/scenario-dialogue/sessions/{id}/audio', {
    method: 'POST',
    body: formData
  });
};
```

**代码示例**：
```go
// 后端：提取生词
func extractVocabulary(turns []models.ScenarioDialogueTurn) []string {
  words := make(map[string]int)
  for _, turn := range turns {
    if turn.Role == "user" {
      // 分词和词频统计
      tokens := tokenize(turn.Content)
      for _, token := range tokens {
        if !isCommonWord(token) {
          words[token]++
        }
      }
    }
  }
  // 返回低频词汇
  return getTopWords(words, 10)
}
```
